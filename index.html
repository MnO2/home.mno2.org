TMonad

a Haskell implementation of L7 cache server

tmonad stands for traffic monad, it is an easy-to-customize HTTP Gateway. tmoand is like Apache traffic server or Varnish, it comes with cache mechanism and routing remap capabilities. Also you can easily custommize it with Haskell programming languages. It follows the design of xmonad, providing hooks for plugins to further customize it.

The motivation of doing this is because most existing reverse proxy application resorts to event-driven programming to pursue for the fastest speed. However, event-driven programming style would become difficult to comprehend when the repository grows large. We believe the powerful GHC runtime with event-driven based IO Manager and threading programming abstraction could resolve this issue, and on the same par with the existing solution on speed.

Encoding Mashup Server

This is a project under Taiwan's Open Data / Netizen movement "g0v". We crawled Chinese dictionary data from goverment's website but it is in different encoding from Unicode.
